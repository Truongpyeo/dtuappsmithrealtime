DTU Appsmith Realtime - Build Instructions

1. Installation:
   npm install

2. Development:
   - Start test server: npm run test
   - Open test/index.html in browser

3. Build:
   npm run build

4. Publishing:
   npm publish

5. Testing CDN:
   - Use version from: https://cdn.jsdelivr.net/npm/dtuappsmithrealtime@1.1.0/dist/index.umd.js
   - Open test/index.html

6. Directory Structure:
   /src
     - index.js
     - websocket.js
     - room.js
   /test
     - index.html
     - test-server.js
   /dist
     - index.umd.js (after build)

7. Version Update:
   - Update version in package.json
   - npm run build
   - git tag v1.0.x
   - git push origin v1.0.x
   - npm publish
